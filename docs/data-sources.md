## 四、选型建议

- **做实盘或高频回测**：优先用 **QMT**，数据本地获取、延迟低、稳定性高。
- **做特征工程、基本面研究、另类数据挖掘**：优先用 **AKShare**，数据种类最丰富且免费。
- **需要标准化、稳定的财务/行情数据**：用 **Tushare**（注意积分限制）。
- **只想批量下载历史日线**：**Baostock** 是零成本入门选择。
- **机构级深度投研**：**Wind / iFinD** 最全，但成本高。

> 以上推荐基于本训练营实践总结，实际使用时请结合具体数据质量、更新频率和接口稳定性综合评估。
